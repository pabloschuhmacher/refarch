Welcome to Customer[0]'s 2019 Ref-arch Beta Refresh (PCFS only)

Thank you for participating in our Beta Refresh! Here is where you'll find the links to the unpublished new refarchs that will eventually fully replace the current ref-arch documentation hosted on docs.pivotal.io.

We are starting incrementally with the new base architecture, AWS and Azure. vSphere refresh will follow. GCP and OpenStack are not currently priorities.

What's the Purpose of this Beta?

Our goal is for the Field to be able to use the new ref-archs with real customers and provide feedback to us. Think of this as our test bed before we publish or make these available to the entire world on docs.pivotal.io. We intend to "bake" the new refarchs by giving PCFS access early and frequently. Your feedback is super important.

How to Access the 2019 Refreshed Beta Ref-Archs

Click on "Ref-Arch Links" in the top navigation. When you click any of the listed refarchs, you will be re-directed to a Pivots-only google doc with comment access where you'll be able to provide feedback directly as you begin using these new docs with customers. Keep in mind: These are works in progress. Check out the "Important FAQs" tab for more information.

How do I Provide Feedback?

For minor changes or copy updates: Provide feedback directly as comments in the ref-arch documents linked here.

For larger requests, usability issues or generalized feedback: Provide feedback using this form.
